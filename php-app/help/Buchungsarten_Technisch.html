CashControl - Buchungsarten Hilfe

Klicken Sie auf einen Schritt im Flowchart, um Details zu sehen.

1. Seite laden und Benutzer prüfen. Nur angemeldete Benutzer dürfen fortfahren.
2. Daten aus der Tabelle buchungsarten laden.
3. DataTables initialisieren (Sortierung, Suche, Paging, responsive).
4. Benutzer klickt "Bearbeiten" oder "Löschen".
5. Bei "Löschen": Modal öffnen → Formular absenden mit CSRF-Token.
6. Buchungsart gelöscht → Toast-Benachrichtigung erscheint.
Anmeldung: Die Seite prüft $_SESSION['userid']. Nicht angemeldete Benutzer werden zu Login.php weitergeleitet.
Datenbank: Buchungsarten werden anhand von userid geladen und in der Tabelle angezeigt.
DataTables: Ermöglicht Paging, Suche, Sortierung und responsive Anzeige. Die ID-Spalte wird verborgen.
Buttons: "Bearbeiten" → EditBuchungsart.php, "Löschen" → öffnet Modal zur Bestätigung.
Modal: Zeigt Löschbestätigung. Formular wird mit CSRF-Token an DeleteBuchungsart.php gesendet.
Toast: Bestätigung für den Benutzer, dass die Buchungsart gelöscht wurde.
